Deploy to GitHub Pages

Option A: Using GitHub Website

  1. Create a new repository named your-username.github.io
  2. Upload all portfolio files to the repository
  3. Go to Settings ‚Üí Pages
  4. Select "main" branch as source
  5. Click Save

Option B: Using Git Commands

# Initialize git repository git init git add . git commit -m "Initial portfolio commit" # Create and push to GitHub git branch -M main git remote add origin https://github.com/your-username/your-username.github.io.git git push -u origin main

To use a custom domain (like yourname.com):

  2. Add a CNAME file to your repository with your domain
  3. Configure DNS settings at your domain provider
